## 1. Master Flow

    class Q1,Q1B,Q2,Q3,Q4,Q5,Q6,Q7,Q8,QPIVOT,S5 decision

## 2. Tool Call Sequence

```mermaid
sequenceDiagram
    participant U as User
    participant N as Neha (Bot)
    participant T1 as get_preferred_institutes
    participant T2 as get_college_recommendations
    participant T3 as search_college_info

    N->>U: Step 1 — 'Deadlines paas aa chuki hain, koi doubt?'
    U-->>N: Reply

    opt Has doubts / no doubts
        N->>T1: Fetch shortlist
        T1-->>N: Shortlist (max 2)
        N->>U: Step 2 / 2A — remind + urgency nudge
    end

    opt Re-engage (pushed back)
        N->>T1: Fetch shortlist
        T1-->>N: Shortlist (max 2)
        N->>U: Step 1B — 'deadlines paas, backup rakh lein'
        Note over N,U: If engages → jump to Step 4
    end

    opt Specific query
        N->>T3: search_college_info(college, fields)
        T3-->>N: Data
        N->>U: Step 3 — answer + WhatsApp + reco nudge
    end

    N->>T2: get_college_recommendations
    T2-->>N: Recommendations (or empty)

    alt Recos exist
        N->>U: Step 4 — pitch 2 colleges + offer deadlines

        opt User asks about a reco
            N->>T3: search_college_info
            T3-->>N: Data
            N->>U: Answer
        end
    else Empty
        Note over N: Skip to Step 6
    end

    N->>U: Step 6 — Counselling handoff<br/>"3–4 forms timely"
    U-->>N: Reply

    alt Accepts
        N->>U: "Experts will call shortly. Have a great day."
    else Declines
        N->>U: Last admission query?
        opt Asks
            N->>T3: search_college_info
            T3-->>N: Data
            N->>U: Brief answer
        end
        N->>U: "Thank you, Have a great day."
    end
```
